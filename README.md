Puzzle Pomodoro
Focus with Pomodoro, unlock puzzle pieces, and reveal your own photo!


## Tech Stack
Frontend: React Native / Expo (TypeScript)

State Management: Zustand

Styling: React Native StyleSheet (Centralized Theme System)

Navigation: Expo Router

## Key Features
Interactive Puzzle Rewards: Earn puzzle pieces for every successful Pomodoro session you complete.

Personalized Experience: Choose your own favorite photos to be revealed as you finish the puzzle.

Daily Progress Tracking: Stay motivated by tracking your focus patterns and daily puzzle achievements.

Themed UI: A warm and intuitive interface designed with a custom color palette and soft shadows for a premium feel.

## Project Structure
/app: Main screens built on Expo Router (Timer, Gallery, Piece-Selection, etc.)

/store: State management logic for timer and puzzle data using Zustand.

/assets: Image and font resources used throughout the app.

theme.ts: Global management of brand colors, spacing, and shadow styles.
